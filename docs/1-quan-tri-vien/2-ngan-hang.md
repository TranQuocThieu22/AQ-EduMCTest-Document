# Module - Ngân hàng câu hỏi

![1](./img/1-ngan-hang/1.jpeg)
Các chức năng của **Module - Ngân hàng câu hỏi**:

**_Quản lí folder:_**

1. Xem danh sách folder
1. Tạo folder
1. Đổi tên folder
1. Xóa folder
1. Xem danh sách folder (chủ đề) trong folder

**_Quản lí câu hỏi:_**

1. Import câu hỏi
1. Xem danh sách câu hỏi trong folder
1. Sửa câu hỏi
1. Xóa câu hỏi
